Q1 なぜプログラミング言語は複数あるのか？

プログラミング言語はコンピューターに命令を与えるためのもの

Write once, run anywhere
一度描けばどこでも動く!

Java登場以前は、利用環境ごとにプログラムを書き直す必要があったが、JavaはPC,サーバー、携帯電話などの様々な場所で動作する

★Javaでプログラムを書いて実行するまでの流れ
1 人間がテキストファイルとしてプログラムを書く
2 JavaコンパイラがテキストファイルをJava用のファイルに変換する(コンパイル）
3 人間には読めないJava用のファイルができる(中間ファイル)
4 Java仮想マシンというアプリケーションが、Java用のファイルを読み取る
5 プログラムとして動作する

Eclipseという統合開発環境を使用する

★JDK(Java Development Kit)
多くの実行ファイルが入っており、特に重要なのはJavaコンパイラとJava仮想マシン

人間用のプログラムの拡張子は「.java」
Java用のファイルの拡張子は「.class」 = 中間ファイル
